# dxtstd-bot

# Installation

## Termux
```cmd
> pkg update && pkg upgrade
> pkg install git -y
> pkg install nodejs -y
> pkg install ffmpeg -y
```

## Cloning this repo
```cmd
> git clone https://github.com/DentaStudio/dxtstd-bot.git
> cd dxtstd-bot
```

## Install the package
```cmd
> npm i
```

## Edit config file
Edit the required value in `config.json`.
```json
{
    "prefix": "%",
    "owner": "6281234567890",
    "botname": "dxtstd-bot"
}
```

## Run the bot
```cmd
> npm start
```
